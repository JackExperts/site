Tecnologias Rancher Labs

K3S

Este é o projeto de Kubernetes Leve, voltado para Edge. Com um único binário com 40 MB o K3S torna possível rodar Kubernetes em plataformas X86 ou ARM com menos de 1 GB de RAM. É uma solução perfeita para soluções rodando em locais remotos e com baixo poder computacional.

RANCHER

Um verdadeiro centro de comando para gerenciar e administrar o seu ambiente Kubernetes em qualquer que seja o provedor de nuvem ou mesmo em on-premises.

RKE

RKE é uma distribuição kubernetes certificada CNCF que roda inteiramente em containers. Uma distro simples, fácil de usar a qual utiliza um único binário para sua instalação. Ela também oferece uma linguagem declarativa para sua configuração e gerenciamento, o que facilita principalmente o processo de upgrade.

LONGHORN

Este é um projeto de storage em bloco distribuído para Kubernetes. Apesar de ter sido desenvolvimento inicialmente pela Rancher Labs, agora ele está sendo mantido pela CNCF. Ele facilita o uso de storage persistente sob demanda dentro do Rancher e Kubernetes de uma forma simples e direta. Permitindo que os times foquem nos projetos e não no storage.
